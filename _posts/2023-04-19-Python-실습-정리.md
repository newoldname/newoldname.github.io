## print()
### 본인의 이름과 생년월일을 화면에 출력
- 요구사항: 생년월일은 년도-월-일 형태로 출력

```python
print("홍길동") 
print("2000-1-1")
```
---
### 본인의 이름과 생년월일을 출력하되, 날짜를 "2000년 1월 1일" 형태로 출력
- 요구사항: print() 명령의 콤마를 사용해서 년/월/일을 분리

```python
print("홍길동") 
print("2000년", "1월", "1일")
```
---
### 본인의 이름과 생년월일을 출력하되, 날짜를 "2000/1/1" 형태로 출력
- 요구사항: 
	- 년, 월, 일은 숫자로 출력
	- 생년월일은 이름과 같은 줄에 출력
	- print() 명령은 분리해서 사용(두 개의 print() 명령 사용)

	
```python
print("홍길동", end = ' ') 
print(2000, 1, 2, sep = '/')
```
