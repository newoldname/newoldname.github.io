## 프로그래밍과 파이썬에 대해서
### 프로그래밍이란
> 프로그램을 계획 or 작업의 순서를 정하거나 실행하는 것.  

또는

> 주어진 문제를 해결하기 위해 작업의 순서를 계획하고 수행하는 것.

그래서 프로그래밍의 핵심이 바로 **요구사항 분석**, 문제를 해결하는 방법 -> **알고리즘**, 마지막 **검수**가 있다. 
사람이 문제를 해석, 요구사항을 분석, 계획하고, 컴퓨터는 사람이 만든 명령들을 처리하면 된다. 

#### 프로그래밍의 저수준 및 고수준

|저수준 Low Leve | <---> | 고수준 High Level|
|:---:|:---:|:---:|
| 하드웨어 종속적 | <---> | 하드웨어에 종속되지 않는 |
|사람보다 기계가 이해하기 쉬운 언어 | <---> | 사람에게 친화적인 언어|
| 어셈블리 언어 | <---> | C, 자바, 파이썬|

#### 프로그래밍의 요소
- 입력: 프로그램이 처리하는 데이터
- 출력: 프로그램에서 처리된 후의 결과중에 사용자에게 보여주는 것.
- 순차: 순서에 따라 처리하는 과정. 일을 순서대로 나눠 수행하는 과정
- 조건: 경우에 따라 구분하여 처리하는 과정
- 반복: 같은 일을 여러 번 실행
- 재사용: 프로그램의 일부를 여러 곳에 갖다 사용하는 것. 

## 파이썬 간단히 알아보기
- 1991년에 만들어지고, 최신 프로그래밍 트렌드에 맞춰 지속적으로 발전해옴. 
- 2.x버전과 3.x버전이 있다. (2버전은 현재 지원 종료됨.)

### 파이썬을 사용하는 이유
- 초보자가 배우기 쉬움. 
- 최신 프로그래밍 기법 지원
- 다양한 분야에서 사용됨. 
- 그렇다고 만능은 아님.

### 파이썬 코드를 짤 때의 주의점
- 파이썬은 UTF-8 형식으로 저장함. -> 그래서 코드에 한글이 들어가 있을 때는 주의해야 함. 
- 파이썬은 대소문자 구분합니다. 그래서 `print()`를 `Print()`로 작성하면 실행할 수 없어요

### 파이썬 코드
```python
print("Hello, World")

# 한 줄 주석은 "#"로 할 수 있다. 

'''
여러 줄 주석은 다운표기호를 3개씩 사용하면 된다. 
'''
```
### 코딩할 때 생길 수 있는 오류
- 문법 오류: 해당 프로그래밍 언어의 문법에 맞지 않는 오류
- 실행 오류: 흔히 버그라고 하는 오류. 문법적으로 오류가 없지만, 실행할 때 이상하게 되는 부분
	- 런타임 오류(runtime error): 컴퓨터가 실행할 수 없는 오류 -> 나누기 0
	- 시맨틱 에러(semantic error): 실행은 문제 없지만, 코드가 잘못 짜서 원하는 결과가 나오지 않는 오류

> 디버그: 실행 중 나타나는 오류를 수정하는 작업

## 파이썬 코딩합시다
## print()
주어진 내용을 화면에 출력. 

```python
print(2)
print(3.4)
print(2+5)
```
### print()함수의 줄바꿈
print함수는 내용을 출력하고 줄을 바꿔준다. 

```python 
print(2); print(3) 
# 출력: 2
# 출력: 3 <-- 중간에 줄바꿈이 일어났다. 
```
### print()함수 파라미터의 콤마는?
print함수 안에 콤마를 사용해 여러 개의 데이터를 출력할 때 중간에 빈칸을 추가하고 이어서 출력한다. 

```python
print("원주율값:", 3.1415)
#  출력--> 원주율값: 3.1415 
# "원주율값:"과 "3.1415" 중간에 빈칸이 들어감.
```

### print()함수의 end옵션
print()함수에 end옵션을 추가하면, 기존에 마지막에 줄바꿈을 했던 작업을 다른 문자/문자열으로 대체 가능

```python
# print(출력_내용, end="다른_문자(열)")

# Terminal에서:
>>> print("홍길동", end=" ")
홍길동 >>>
>>> print("홍길동", end = " (끝)") 홍길동 (끝)>>>
>>> print("홍길동", end = " (끝)\n") 홍길동 (끝)
>>>
```
### print()함수의 sep옵션
print()함수에 sep옵션을 추가하면, 기존에 콤마로 구분한 인자들 사이에 빈칸으로 채워진 것은 특정 문자/문자열으로 채워지게 된다. 
> end와 sep는 동시에 사용 가능

```python
# print(출력_내용, sep="다른_문자(열)")

# Terminal에서:
>>> print("홍", "길동")
홍 길동 
# 기본적으로 중간에 빈칸으로 구분됨.
>>> print("홍", "길동", sep = "\n")
홍
길동 
# 이제는 줄바꿈 문자로 채워짐.
>>> print("성: 홍", "길동", sep = "이름") 
성: 홍 이름: 길동 
# 지금은 중간에 "이름"으로 구분하게 됨.
>>> print("홍", "길", "동", sep = "\n")
홍
길
동 
# 모두 줄바꿈 문자로 대체됨.
```

## 파이썬의 자료형
컴퓨터는 안녕하세요를 문자열으로, 3.14를 숫자로 구분하게 만드는 것이 바로 자료형이다. 
### 자료형의 역할
- 값의 종류 및 표현 가능한 범위를 정해주고
- 메모리에 값을 저장하는 방식을 정하고
- 사용 가능한 연산자를 제한하고
- 연산결과의 종류를 알 수 있게 한다. 


### 파이썬에서의 자료형이
실행하면서 결정된다. 

> 따움표가 있으면 문자열
> 숫자만 있으면 정수 또는 실수



## 자료형의 종류
### 정수형 int
- 소수점 없는 숫자
- 파이썬에서는 int형으로 표기
- 표현 범위 제한 없음(파이썬에서)

### 실수형 float
- 부동 소수점으로 저장
- 미세한 오차 발생 가능

### 문자열
- 한 쌍의 따움표 안의 있는 글자들
- 여러 줄의 문자열은 

```python 
a = """
이렇게 합니다. 
"""
```
> 여러 줄 주석 처리와여러 줄 문자열 처리의 방식은 같지만, 여러 줄 문자열은 앞에 변수에 값을 할당하는 부분이 있고, 주석 처리는 따로 변수에 값을 할당하지 않는다. 

### 특수 문자
| 특수 문자 | 설명|
| :--: | :--:|
| \n | 줄바꿈 문자 |
| \t | 탭Tab문자(위아래 문자들의 간격을 맞출 때 사용 가능) |
| \\ | 백 슬래시(반대인 슬래시) |
| \' | 작은 따움표(따움표 자체를 하나의 문자열으로 인식하고 싶을 때 사용) |
| \" | 큰 따움표(따움표 자체를 하나의 문자열으로 인식하고 싶을 때 사용) |

> 인코딩
> 이러한 문자들은 컴퓨터 안에서는 ~~당연히~~ 0과 1의 조합으로 저장되어 있고, 문자를 0과 1의 조합으로 변환하는 것이 인코딩입니다. 파이썬은 `UTF-8` 방식을 사용합니다. 


### Bool자료형
- 참 또는 거짓을 구분하는 자료형. 
- 파이썬에서는 `True`, `False`로 사용 가능

### None상수값
- NoneClass를 나타나는 상수값이다. -> None 값을 제공하기 위함.


> None의 사용 목적
>
>- 값이 존재하지 않음을 명시함.
>- 값을 정확하게 알 수 없을 경우 표현
>- 값이 정의되지 않은 경우 표현 

### 리터럴(Literal) 상수
코드에서 직접적으로 명시되는 숫자값이나 문자열

### 상수
- 프로그램이 실행 중에 변하지 않는 값
- 코드에 사용된 숫자, 문자열들 포함

값이 어떤 자료형인지 확인가능

## int(), float(), str() 명령
### int()
`실수값` 또는 `수소점이 없는 숫자`로만 구성된 문자열을 정수로 변환해서 반환

```python
>>> int("123") # 정수 문자열을 정수로 변환
123
>>> int(123.6) # 소수점 이하 버림 (반올림 안함) 123
```
잘못 사용되는 경우 -> 정수가 아닌 문자열

```python
>>> int("123.0")
Traceback (most recent call last):
File "<stdin>", line 1, in <module> ValueError: invalid literal for int() with base 10: '123.0'
```

### float()
`정수값` 또는 `숫자들로 구성된 문자열`을 실수로 변환해서 반환

```python
>>> float("123.2") # 실수형 문자열을 변환
123.2
>>> float("123") # 정수형 문자열도 실수값으로 변환 
123.0
>>> float(123) # 정수를 전달해도 실수값으로 변환 
123.0
```
잘못 사용되는 경우 -> 숫자가 아닌 문자열

```python
>>> float("123.0a")
Traceback (most recent call last):
File "<stdin>", line 1, in <module> ValueError: could not convert string to float: '123.0a'

>>> float("a123.0")
Traceback (most recent call last):
File "<stdin>", line 1, in <module> ValueError: could not convert string to float: 'a123.0'
```
###  str()
주어진 값을 문자열로 반환

```python
>>> str(123.4) 
'123.4'
>>> str(123) 
'123'

# Bool값을 문자열으로 반환 가능
>>> str(True) # 불린값을 문자열로 변환 
'True'
>>> str(3 < 2)
'False'
```

## 표현식(expression) vs 명령문(statement)
- 표현식(expression)
	- 값을 만들어내는 코드
	- 상수, 연산 결과, 변수, 함수 호출 등 포함
- 명령문(statement)
	- 컴퓨터에게 작업을 수행하라고 명령하는 코드
	- 대입문, 조건문, 반복문 등 있음.


## 산술 연산
- 정수/정수 연산자 끼리의 연산 결과는 정수
- 그 이외의 연산결과는 실수임.
- `+`, `-`, `*`, `/(실수 나눗셈)`, `//(정수 나눗셈)`, `%(정수 나머지)`, `**(제곱)` 등 있다. 
### 연산 순서
1. 괄호 `()`
2. 제곱 `**` -> `2**3 = 8`
3. 곱셈, 나눗셈 `*`, `/`, `//`, `%`
4. 덧셈, 뺄셈 `+`, `-`

### 문자열의 덧셈, 곱셈
문자열들을 더하면 합쳐진 새로운 문자열 탄생 -> `"안" + "녕" -> "안녕"`

문자열들을 곱하면 합쳐진 새로운 문자열을 곱한 횟수 생성 -> `"안녕" * 3 -> "안녕안녕안녕"`

> 문자열의 덧셈, 곱셈도 같은 우선순위 있음 -> `괄호 --> 곱셈 --> 덧셈`

## 변수
### 변수가 왜 필요해?
연산 결과를 저장하기 위함. 
> 간단한 하드웨어 설명을 해보자면
> CPU, 프로세서: 계산 담당
> 메모리: 컴퓨터의 기억 공간, 데이터를 임시적으로 저장
> 보조 기억 장치: 파일 등 자료를 저장

### 그래서 변수의 정의
값을 담아두는 메모리 공간 or 메모리 공간에 저장된 값
### 대입연산자 "="
변수는 대입연산자 기준으로 왼쪽에 있으면 메모리 공간, 오른쪽에 있으면 메모리 공간의 값을 의미함.

### 변수의 이름 짓기 규칙
- 이름 길이 제한 없음.
- 변수는 반드시 유니코드 "문자" 또는 _로 시작해야 함. 
- 변수 이름에 공백이 있으면 안됨. 
- 특수 문자 활용 불가
- 키워드/예약어 사용 불가
- 대문자/소문자 구분한다.

### 복합연산자
`+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=`등 있다. 

### 형식 지정자
```python
# %d, %s, %f등 사용 가능

>>> s = "원의 반지름: %d" % 20 # 문자열 생성 후 출력 
>>> print(s)
원의 반지름: 20

>>> print("원의 반지름: %f" % 20) # print( ) 명령에 생성되는 문자열을 바로 전달
원의 반지름: 20.000000

>>> print("너비가 %d이고 높이가 %d인 사각형의 넓이 는 %d이다" % (20, 30, 20 * 30))
너비가 20이고 높이가 30인 사각형의 넓이는 600이다

# 소수점 자릿수 지정 가능 (반올림도 해줌)
>>> print("원주율: %.3f" % 3.1415) 
원주율: 3.142

# 전체 자릿수도 지정가능 -> 빈칸으로 자리를 확보
>>> print("%10s" % "한글") 
>>>        한글
```

## input()
키보드로 입력을 받는 것. 
프룸프트가 생길 때 키보드로 입력하면 됨.

입력을 받을 때 사용자에게 보여진다. 
> 하지만 input로 받은 데이터는 모두 문자열이기 때문에 숫자를 받고 싶으면 따로 int(), float()함수를 사용해야 한다.

```python
# (1) 실수 한 개를 입력하세요.입력한 값에1을 더한 후 출력합니다:

```
## 조건문, 비교연산자, 논리 연산자
### 비교 연산자
`==`, `!=`, `>=`, `>`, `<=`, `<`, 등 있다. 
### 문자열 비교
- 같은 언어 비교 시 사전순으로 비교
- 영어는 한국어보다 작다
- 대문자 알파벳이 소문자 알파벳보다 작다. 
- 특수 기호와 숫자는 알파벳보다 작다. 

```python
>>> "한글" == "한글"
True
>>>"가"<"나"    # 같은 언어 비교시 사전 순
True
>>> "English" == "English"
True
>>> "English" < "Korean"    # 같은 언어 비교시 사전 순 
True
>>> "English" < "한글"    # 영문 < 한글
True
>>> "English" > "english"      # 영문 대문자 < 영문 소문자 
False
>>> "1234" < "A" # 숫자 < 알파벳
True
```

### 숫자 비교
- 정수 및 실수는 서로 비교 가능
- bool자료형에서 `True == 1`, `False == 0`

```python
>>> 2 == 2.0 
True
>>> 2 < 2.1 
True
>>> 2.1 >= 2.3
False

>>> True == True 
True
>>> True == False
False
```

### 다른 자료형 끼리의 비교
자료형이 같은 지 `==`, `!=`으로 비교가 가능하지만, 대소 비교는 `Error`


### 조건식에 대해
- 숫자 들어가면: 0 == False, 나머지 모두 True로 간주한다.
- 문자열`""(비어 있는 문자열)`이면 False, 나머지 모두 True
- 리스트: 빈 리스트는 False, 나머지는 모두 True
- 튜플: 빈 튜플은 False, 나머지는 True
- 딕셔너리: 비어 있으면 False, 아니면 True

> 튜플은 변경불가인 리스트 -> 소괄호로 묶어서 하기
> 
> 딕셔너리는 key와 value을 한 쌍으로 구성된 특별한 리스트이다. -> 중괄호로 묶기

### if ~~~ elif ~~~ else 사용 가능

### if문의 단락평가
- 조건문 and의 앞부분만 Flase가 뜨면 뒷부분은 쳐다보지도 않고 바로 Flase 반환함.
- 효율을 위함.

### pass문
아무거도 하지 않는다. 

```python
if a > 5:
    print(a)
else:
    pass # 아무것도 안하지만, else 부분의 코드 블록을 구성하고 있다.
```

## 논리 연산자
`and`, `not`, `or`가 있다. 

## 각종 연산자의 우선 순위
산술/비교/논리/대입 등 연산자
1. 괄호: `( )`
2. 거듭 제곱: `**`
3. 곱셈, 나누셈: `*`, `/`, `//`, `%`
4. 덧셈, 뺄셈: `+`, `-`
5. 대소 비교연산자: `<`, `<=`, `>`, `>=` 
6. 일치 비교 연산자: `==`, `!=`
7. 대입 연산자: `=`, `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=`
8. 논리 연산자: `and`, `or`, `not`
